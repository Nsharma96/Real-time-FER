# Real-time-FER
Facial Expression recognizer uses dnn to train learning model on fer2013 data set(kaggle) then uses opencv to load model capture video and make real time predictions 

## Model result after training
- loss: 0.4036, 
- accuracy: 0.8679,
- val_loss: 1.9367,
- val_accuracy: 0.6090, 
- lr: 1.0942e-04
